## 🚀 使用方式

### 快速开始
```bash
# 运行演示程序
python quick_demo.py

# 获取实时价格（如网络允许）
python main.py --mode realtime --codes 000001 600519

# 获取历史数据并保存
python main.py --mode historical --save
```

### 编程接口
```python
from data_fetcher import StockDataFetcher

fetcher = StockDataFetcher()

# 获取历史数据
hist_data = fetcher.get_historical_data("000001")
print(f"获取到 {len(hist_data)} 条记录")

# 获取基本信息
info = fetcher.get_stock_info("000001") 
print(f"股票名称: {info['股票简称']}")
```
