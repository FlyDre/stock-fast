# 股票数据获取工具 - 图形界面使用说明

## 🖥️ 界面版本介绍

### 完整版界面 (`stock_ui.py`)
- **功能完整**：支持批量处理、多种数据类型获取
- **界面尺寸**：800×700，适合详细操作
- **适用场景**：专业数据获取、批量处理

### 简化版界面 (`simple_ui.py`)  
- **操作简单**：单股票查询，一键获取
- **界面尺寸**：600×500，简洁紧凑
- **适用场景**：快速查询、学习使用

## 🚀 启动方式

### 方法一：双击启动（Windows）
```
双击文件：启动界面.bat
```

### 方法二：命令行启动
```bash
# 完整版界面
python stock_ui.py

# 简化版界面  
python simple_ui.py
```

## 📊 完整版界面功能说明

### 1. 股票代码设置区域
- **输入框**：支持多个股票代码，用逗号分隔
- **默认按钮**：一键加载配置文件中的默认股票
- **示例格式**：`000001,002594,600519`

### 2. 功能选择区域
- **基本信息**：获取公司名称、市值、财务指标
- **历史数据**：获取K线数据、交易量等历史信息
- **全部数据**：同时获取基本信息和历史数据
- **保存选项**：自动保存数据到CSV文件

### 3. 操作按钮
- **开始获取**：启动数据获取过程
- **清空结果**：清除显示区域内容
- **打开数据目录**：直接打开保存数据的文件夹

### 4. 状态显示
- **状态栏**：显示当前操作状态
- **进度条**：显示处理进度（动画效果）
- **结果区域**：实时显示获取过程和结果

## 🎯 简化版界面使用

### 操作步骤
1. 在输入框中输入6位股票代码（如：000001）
2. 点击"查询"按钮或按回车键
3. 在结果区域查看股票信息

### 显示内容
- **基本信息**：公司名称、市值、市盈率等
- **最近交易**：近5天的价格和成交量数据
- **格式清晰**：表格化显示，易于阅读

## 🔧 常见操作

### 批量查询股票
1. 在股票代码框输入：`000001,002594,600519,300750`
2. 选择"获取全部数据"
3. 勾选"保存数据到CSV文件"
4. 点击"开始获取"

### 快速查看单只股票
1. 使用简化版界面：`python simple_ui.py`
2. 输入股票代码，如：`000001`
3. 按回车键或点击查询

### 查看保存的数据
1. 点击"打开数据目录"按钮
2. 或手动打开项目目录下的 `data/` 文件夹
3. 使用Excel或其他工具打开CSV文件

## 💡 使用建议

### 网络环境
- **历史数据**：通常可正常获取
- **基本信息**：稳定可用
- **实时价格**：可能受网络代理影响

### 批量处理
- 建议一次处理不超过10只股票
- 大量数据获取建议分批进行
- 注意观察状态提示信息

### 数据保存
- CSV文件支持Excel直接打开
- 文件名包含时间戳，避免覆盖
- 数据格式标准化，便于分析

## 🚫 故障排除

### 界面无法启动
```bash
# 检查Python环境
python --version

# 检查依赖包
pip install -r requirements.txt
```

### 数据获取失败
- 检查网络连接
- 确认股票代码格式正确
- 查看状态栏错误信息

### 文件保存问题
- 确保有写入权限
- 检查磁盘空间
- 关闭占用CSV文件的程序
